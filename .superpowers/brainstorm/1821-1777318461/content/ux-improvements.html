Which UX improvements do you want?

Select all that feel right — these are the biggest quality-of-life wins for the scene maker.

A

Undo / Redo (Ctrl+Z)

Right now one wrong click ruins a wall run. Undo makes building much less frustrating.

B

Object Rotation (R key)

Cycle placed objects through 0° / 90° / 180° / 270°. Essential for furniture and doors.

C

Ghost Preview Before Placing

Show a translucent version of the selected asset under the cursor before you click. You always know what you're about to place.

D

Save / Load Scene

Scenes are lost on page refresh. Save to the database per group, load back anytime. Already planned in the backlog.

E

Better Fill Tools

Fill Floor already exists, but add Fill Room (walls around a rectangle) and Clear Section (erase a dragged area) for faster scene building.

F

Asset Search / Filter

203 models in a list is hard to navigate. Add a search box at the top to filter by name instantly.
